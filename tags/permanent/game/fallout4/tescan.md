## Unable to download 'TesCan' files

If your manual helper file or log file shows references to 'TesCan' files failing to download,
i.e. `Unable to download Data_ccBGSFO4046-TesCan.esl`

Follow these steps:

1. Navigate to your Steam's Fallout 4's Data folder.
2. Within the data folder, locate the file, `ccBGSFO4046-TesCan.esl`. and make note of whether the file name is ccbgsfo4046-tescan or ccBGSFO4046-TesCan.  Pay attention to the upper or lowercase in TesCan.
3. Delete the file.

If the file was called `ccbgsfo4046-tescan.esl` with a lower case tescan
1. Open Steam.
2. Right click on Fallout 4. Select Properties.  Select Installed Files.  Click on "Verify integrity of game files".
3. Once the files have been verified, go back to your Fallout 4's Data folder and confirm that you now have ccBGSFO4046-TesCan.esl that has an upper case C.
4. Rerun the Wabbajack installation.

If the file was called `ccBGSFO4046-TesCan.esl` with an upper case TesCan
1. Launch Fallout 4 from Steam.
2. Select Creations from the main menu.
3. Select Category at the bottom, or press Shift.
4. Select the category Creation Club.
5. Find "Tesla Cannon" in the alphabetically sorted list.
6. Download Tesla Cannon.
   **DO NOT Alt+Tab; you must leave the game window active or the download will stop.**
7. Exit the game.  DO NOT perform a verify in Steam.
8. Go back to your Fallout 4\Data folder and verify that you now have ccbgsfo4046-tescan.esl that has a lower case c.
9. Rerun the Wabbajack installation.
